Secret Message Box

Create a special passcode and store it in a String variable

Using a pop-up, ask the user for a secret message and store it in a variable

Now ask them to guess the passcode.

If their guess matches the special passcode, show them the secret message

If the passcode does not match, pop-up "INTRUDER!!"

[optional] Have them also enter a username, and make sure it is also correct before releasing the secret message.
