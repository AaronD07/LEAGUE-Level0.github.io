The League of Amazing Programmers
Level 0 Overview - For Teachers

INTRO TO JAVA WORKSHOP
(Eclipse)

Introduction to Java programming for students not yet enrolled at The League.
Note: Some workshops are compressed into 2 longer days.

  • Day 1 - Processing: make simple sketches (int).
  • Day 2 - Eclipse: Use Robot to draw shapes.
  • Day 3 - Eclipse: Use dialogs to interact with user (String).
  • Day 4 - Eclipse or Processing: Complete recipe projects.
  • Day 5 - Eclipse: Scary Maze project.

    • MODULE 0 - Moving the Robot, using String variables
    Eclipse - bit.ly/l0m0test

    Recipes for students who have just started Level 0.
    At the end of this module, students will be able to:

  • Create a new Java class
  • Add a main method to the class
  • Code a pop-up dialog to display a message to the user
  • Code a pop-up dialog to ask the user for their favorite color
  • Join Strings using +
  • Use the Robot class to draw a shape

    • MODULE 1 - conditionals, variables, loops
    Processing - bit.ly/l0m1proc
    Eclipse - bit.ly/l0m1test

    Recipes for students who have passed the module 0 checkpoint.
    At the end of this module, students will be able to:

  • Use an if statement to control program flow
  • Create and initialize variables
  • Use a for loop to repeat blocks of code
  • Use int variables in simple math statements
  • Know where curly brackets {} are required in a simple program
  • Draw simple graphics using Processing
    • MODULE 2 - calling methods, Random numbers, else
    Processing -bit.ly/l0m2proc
    Eclipse - bit.ly/l0m2test

    Recipes for students who have passed the module 1 checkpoint.
    At the end of this module, students will be able to:

  • Use the Random class to generate random integers
  • Convert String variables to int for math operations
  • Use else statements to further control program flow
  • Call simple methods to match their method signature

    • MODULE 3 - else if, more loops, modulo
    Processing - bit.ly/l0m3proc
    Eclipse - bit.ly/l0m3test

    Recipes for students who have passed the module 2 checkpoint.
    At the end of this module, students will be able to:

  • Code more complex logic flow using else if conditionals
  • Use the modulo operator in conditionals and loops
  • Call multi-parameter methods to match their method signature
  • Write simple methods

    • MODULE 4 - nesting conditionals and loops
    Processing
    Eclipse - bit.ly/l0m4test

    Recipes for students who have passed the module 3 checkpoint.
    At the end of this module, students will be able to:

  • Code more complex logic flow by nesting conditionals and loops
  • Demonstrate readiness for Level 0 exam via practice tests
